Projects

  • Forever Glow: Forever Glow was a website created that helps empower young women by recommending products that will help elevate their appearance and glow up to feel confident and beautiful. This website was created using HTML, CSS, and JavaScript. I worked on this project with two other classmates in my Computer Science Essentials class.
  • EduConnect: EduConnect is a website that combines a student's love with social media with their need to prioritize their studies. I created this website along with two of my other friends in a Hackathon to provide a solution that improves students' education by creating an app that acts similar to a social media platform with its' content tailored to the student's needs as determined by their teacher.
  • Parent Report: Parent Report was an app created by my friend and I to help teachers and students in the grading system. Teachers often struggle to grade all of the tasks they assign and fail provide feedback in a reasonable period. Therefore, we created a solution by having parent volunteers use this app to grade students' work anonymously such that they understand how a teacher grades, the teacher has some stress aleviated, and students recieve timely feedback in order to make adjustments.
  • Tic Tac Toe: This is a Tic-Tac-Toe game coded in Pythnon and PyGame. I coded this project with my father and was responsible to work on the front-end and back-end of the project. I used the Pygame library to create the game and the Python library to create the front-end. I learned how to use Pygame and create graphics.
